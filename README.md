# rembg-server

A docker container for running [**rembg**](https://github.com/danielgatis/rembg) as a server instance!

## Usage

```sh
docker run -d -v $HOME/.u2net:/root/.u2net -p 5000:5000 --name rembg-server muelheimmodular/rembg-server
```
